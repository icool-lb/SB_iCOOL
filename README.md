# 🥇 Silver Bullet Gold — نظام تداول XAU/USD احترافي

نظام تداول متكامل لاستراتيجية **Silver Bullet (ICT)** على الذهب، يتضمن:

- ⏰ **كاشف النوافذ الزمنية** الثلاث (London / NY AM / NY PM) بتوقيت نيويورك تلقائياً
- 📊 **محرك إشارات آلي**: يكتشف أخذ السيولة + Fair Value Gap + هيكل السوق ويحسب الدخول/الوقف/الهدف
- 🤖 **تحليل ذكي** للإعداد عبر OpenAI (GPT-4o)
- 📝 **سجل صفقات كامل** مع فتح/إغلاق الصفقات
- 📈 **إحصائيات الأداء**: Win Rate، صافي الربح، Profit Factor، التوقع، منحنى رأس المال
- 💾 **حفظ عبر كل أجهزتك** (Supabase) أو محلياً في المتصفح
- 🌙 واجهة عربية RTL داكنة احترافية، تعمل على الموبايل

---

## 🚀 خطوات النشر على GitHub + Vercel

### 1) ارفع المشروع على GitHub
- أنشئ مستودعاً جديداً (مثلاً `silver-bullet-gold`)
- ارفع كل الملفات إليه

### 2) احصل على المفاتيح المطلوبة

**MetaApi (إلزامي لسعر الذهب الحي):**
1. سجّل في https://app.metaapi.cloud
2. اربط حساب MT4/MT5 (يكفي حساب تجريبي Demo)
3. من **Token Management** أنشئ توكناً → انسخه إلى `METAAPI_TOKEN`
4. انسخ **Account ID** للحساب → إلى `METAAPI_ACCOUNT_ID`

**OpenAI (للتحليل الذكي):**
- من https://platform.openai.com/api-keys أنشئ مفتاحاً → `OPENAI_API_KEY`

**Supabase (اختياري — لحفظ الصفقات عبر أجهزتك):**
1. أنشئ مشروعاً مجانياً على https://supabase.com
2. من **SQL Editor** الصق محتوى ملف `supabase_schema.sql` وشغّله
3. من **Project Settings → API** انسخ:
   - `Project URL` → `SUPABASE_URL`
   - `service_role key` → `SUPABASE_SERVICE_KEY`
> إن تركت Supabase فارغاً، ستُحفظ الصفقات محلياً في متصفحك فقط.

### 3) انشر على Vercel
1. ادخل https://vercel.com → **Add New Project**
2. اختر مستودع GitHub
3. في **Environment Variables** أضف المتغيرات من ملف `.env.example`
4. اضغط **Deploy** — خلال دقيقتين سيكون نظامك جاهزاً على رابط `https://...vercel.app`

## 💡 كيف يعمل النظام؟

1. يجلب سعر الذهب والشموع (5 دقائق) من MetaApi كل 30 ثانية
2. يتحقق إن كنا داخل نافذة Silver Bullet
3. يبحث عن: أخذ سيولة (قاع/قمة) + FVG غير ممتلئ في اتجاه الدخول
4. يولّد إشارة بمستويات دخول/وقف/هدف (R:R = 1:2 افتراضياً)
5. تضغط "فتح صفقة" → تُسجّل، ثم تغلقها على الهدف أو الوقف
6. تُحدّث الإحصائيات و Win Rate ومنحنى رأس المال تلقائياً

## ⚙️ تشغيل محلي (اختياري)
```bash
npm install
cp .env.example .env.local   # املأ المفاتيح
npm run dev                  # http://localhost:3000
```
